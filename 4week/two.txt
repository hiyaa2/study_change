<가위바위보>

경우의 수 : 바위,가위,보자기

1.승리조건을 정한다
-바위를 냈을 때 가위가 나오면 승리
-가위를 냈을 때 보자기가 나오면 승리
-보자기를 냈을 때 가위가 나오면 승리

2.같은 경우의 수를 냈을때를 정한다
-바위를 냈을 때 바위가 나올 때
-가위를 냈을 때 가위가 나올 때
-보자기를 냈을 때 보자기가 나올 때

3.가위보위보를 실행한다.

4. 바위일때 가위, 가위일 때 보자기, 보자기 일 때 바위가 나오면 승리한다

<묵찌빠>

경우의 수 : 바위,가위,보자기

1. 가위바위보를 진행한다. (가위바위보의 컴퓨팅적 사고에 의존하여)

2. 가위바위보에서 승리한 사람이 묵찌빠의 공격권을 갖는다.

3. 1에서 진행한 가위바위보를 이긴 사람이 처음 묵찌빠의 공격권을 갖게 되며
비길 때 까지 진행한다.

4. 같은 손 모양 일 때 승리한다.
ex) 같은 바위, 같은 가위, 같은 보자기 일 때
